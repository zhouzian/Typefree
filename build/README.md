# Build Resources

This directory contains build resources for electron-builder.

## Assets

All assets are generated by running:
```bash
npm run generate-assets
```

### Icons

- `icon.svg` - Master SVG icon (1024x1024)
- `icon.icns` - macOS icon bundle (auto-generated)
- `icon.ico` - Windows icon (auto-generated)
- `icons/` - PNG icons in various sizes
  - `icon-16.png` through `icon-512.png`
  - `tray.png` - System tray icon (normal state)
  - `tray-recording.png` - System tray icon (recording state)

### Sounds

Located in `src/assets/sounds/`:
- `start.wav` - Subtle ascending tone when recording starts
- `end.wav` - Subtle descending tone when recording ends

The sounds are non-intrusive, with soft sine wave tones and fade in/out.

## Design

### App Icon
- Modern gradient background (purple/indigo)
- Clean microphone silhouette
- Rounded corners (20% radius)
- Subtle shadow effect
- Decorative accent circles

### Tray Icon
- Compact 22x22 pixel design
- Same microphone motif
- Purple gradient for normal state
- Red gradient for recording state

## Manual Generation

If you need to regenerate assets individually:

```bash
# Generate sounds
npx tsx scripts/generate-assets.ts

# Generate PNG icons
npx tsx scripts/generate-icons.ts

# Create Windows .ico
npx tsx scripts/create-ico.ts

# Create macOS .icns (requires iconutil on macOS)
iconutil -c icns build/icon.iconset -o build/icon.icns
```
